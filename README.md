# Synthetic-simulation

本项目基于 MATLAB，旨在实现和展示合成模拟（Synthetic Simulation）相关的算法与应用。


## 主要内容

- 合成数据的生成方法
- 仿真过程的实现和结果展示
- 可根据需求拓展不同类型的模拟应用

## 环境要求

- MATLAB R2018b 及以上版本

## 快速开始

1. 克隆本项目

   ```bash
   git clone https://github.com/xsmsily/Synthetic-simulation.git
   ```

2. 在 MATLAB 中打开项目文件夹

3. 运行主脚本

   ```matlab
   main_simulation
   ```
